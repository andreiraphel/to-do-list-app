# to-do-list-app

Task Manager - Simple To-Do List Application
A straightforward Java-based task manager that helps you organize your tasks with descriptions, deadlines, and priority levels.
Features

Add Tasks: Create new tasks with detailed descriptions
Set Deadlines: Assign a year to each task for tracking
Priority Management: Mark tasks with priority levels (High, Medium, Low)
Delete Tasks: Remove completed tasks from your list
Simple Interface: Easy-to-use command-line interface
